# 📊 Data Analysis Projects: Flights EDA & Customer Analysis Dashboard

This repository brings together two different data analysis projects that focus on extracting meaningful insights from real-world datasets using both Python and visualization tools. While the first project explores flight data through exploratory data analysis (EDA), the second project presents customer behavior insights using an interactive dashboard built in Tableau. Together, these projects reflect a practical approach to understanding data from both analytical and visual storytelling perspectives.

The Flights EDA project focuses on analyzing airline data to uncover patterns related to delays, cancellations, and operational performance. Using Python libraries such as Pandas, Matplotlib, and Seaborn, the dataset was cleaned, explored, and visualized to identify trends like peak delay times, frequently affected routes, and overall flight performance. This project helped in understanding how raw data can be transformed into meaningful insights through systematic analysis and visualization.

The Customer Analysis project, on the other hand, shifts focus toward business intelligence. Built using Tableau, this dashboard provides a visual representation of customer-related metrics such as purchasing behavior, segmentation, and overall trends. The goal was to create an intuitive and interactive interface that allows users to explore the data easily and derive insights without needing technical knowledge. This highlights the importance of presenting data in a way that is both accessible and impactful.

Together, these projects demonstrate a balance between technical data analysis and visual storytelling. They showcase skills in data preprocessing, exploratory analysis, and dashboard creation, while also emphasizing the importance of communicating insights effectively. These projects serve as a foundation for working with more complex datasets and building end-to-end data-driven solutions in the future.
